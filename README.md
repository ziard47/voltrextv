# Voltrex TV

Voltrex TV is a futuristic, web-based IPTV player designed for the modern age. It allows users to watch thousands of free-to-air TV channels from around the world in a sleek, neon-infused interface.

## Features

*   **Modern UI:** A visually stunning interface with neon accents and glassmorphism effects.
*   **Live TV:** Stream thousands of channels directly in your browser.
*   **Country Filtering:** Browse channels by country (Sri Lanka featured by default).
*   **Search:** Quickly find your favorite channels with a built-in search function.
*   **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.
*   **HLS Support:** Built-in support for HLS (m3u8) streams.

## Setup & Usage

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/voltrex-tv.git
    ```
2.  **Open the project:**
    Navigate to the project directory and open `index.html` in your web browser.
3.  **Enjoy:**
    Start watching your favorite channels!
